# Modern Portfolio Website

## 🚀 Quick Start

Your modern portfolio website is ready! 

### Preview
Simply open `index.html` in your browser:

```bash
# Windows
start index.html
```

```bash
# Or use a local server for best experience (recommended)
npx serve .
```

### Features ✨
- 📱 Fully responsive design (mobile-first)
- 🌙 Dark/Light mode toggle (persists in localStorage)
- 🎨 Modern glassmorphism design with smooth animations
- 📊 Hero, About, Projects, Contact sections
- ⚡ Smooth scrolling navigation
- 📝 Working contact form (client-side validation)
- 🎭 Scroll-triggered animations
- 🪂 Parallax effects

### Customization 🔧
Edit these files:
- `index.html` - Content, projects, bio
- `styles.css` - Colors, fonts (CSS custom properties)
- `script.js` - Functionality

Replace placeholder images with your own in the projects section.

### Deployment 🚀
1. **GitHub Pages**: Go to repo Settings > Pages > Deploy from branch `main`
2. **Netlify/Vercel**: Drag & drop the folder

Enjoy your new portfolio! 🎉
